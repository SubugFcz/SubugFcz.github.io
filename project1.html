Temperature and Humidity Sensor

The problem with my dorm is we don't have any measurement tools to know how cold or hot it is inside our dorm room. Sometimes, it took a lot of work for us to set our temperature to the sweet spot that we wanted. If we use the weather app on our phone, it is the whole city's temperature. Meanwhile, what I want is the local temperature inside my dorm. So we need tools to measure temperature. As I have the resources to build my own temperature and humidity sensor, I started this project.

Temperature and Humidity Sensor that measures accurately within 0-50 degrees Celcius and 5-95% humidity range.

The Arduino itself can be reusable for different purposes, and I can borrow other stuff that is needed in this project from my GT ECE store. Thus, this project has become affordable and fun for learning purposes.

  1. I started by getting all the things ready. Things that I need:

Arduino Uno

Breadborad

Male jumper wires

Male-to-female jumper wires

16x2 LCD screen with I2C adapter

DHT11 Sensor

10k ohm through-hole resistor

9VDC 1A Power Supply Adapter 110V for Arduino Uno

  1. Connecting the parts according to the schematics
  1. Code! See my code here

An improvement that can be made for this project includes a better sensor (DHT22) and housing or mounting for the whole components. It might be 3d printed, or just simple cardboard cutting will work.

Definitely a fun entry project to do and takes a small amount of time at an affordable cost.
